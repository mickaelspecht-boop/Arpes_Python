# Shortcuts

## Navigation

- Left arrow: load the previous file in the browser.
- Right arrow: load the next file in the browser.
- Double-click file: load that file.
- Double-click group: expand or collapse the group.
- Ctrl+O: open a saved session.
- Ctrl+Shift+S: save the current session as a chosen file.

## Map Reading

- Left-click on a map: move the current MDC/EDC probe to that energy and momentum.
- Mouse wheel or Matplotlib toolbar: zoom and pan plots.
- FS Explorer line center: drag to move the cut.
- FS Explorer line handle: drag to rotate or resize the cut.

## Fitting and Cleanup

- Right-click a kF point: add or edit an annotation.
- Select kF points: mark points for deletion from the fit controls.
- Ctrl+Z: undo the last kF point deletion.
- Ctrl+Y: redo the undone deletion.

## Organisation

- Tags field: add free labels to the current file, separated by commas.
- Tag filter: show only files carrying the selected tag.
- Notes tab: write session decisions in Markdown, then use Preview to read them.

## Useful Menus

- File > Recent sessions: reopen a recent analysis.
- File > Compare sessions: compare two saved sessions.
- Logbook > Load logbook: attach metadata to the whole folder.
- Logbook > Add subfolder-scoped logbook: attach metadata only to one sample folder.
- Cache > Reload current file (no cache): force a fresh load after metadata or file changes.
